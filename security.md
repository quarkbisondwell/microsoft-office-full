1. 检查是否有多余空格，直接复制粘贴

## ❓ 常见问题

**❓ 这是真的完整版吗？**  
是的。包括 Word、Excel、PowerPoint、Outlook、Access、Publisher、OneNote 全部组件。

**❓ 和 Office 365 有什么区别？**  
Office 365 是按年/月订阅，有云存储和持续更新。这个是永久授权版（批量许可版），功能相同，只是没有 OneDrive 云空间送。

**❓ 支持 Mac 吗？**  
这个版本是 Windows 版。Mac 用户需要专门的 Mac 版 Office。

**❓ 需要联网才能用吗？**  
不需要。激活后完全可以离线使用。

**❓ 安全吗？**  
是的。软件原汁原味来自微软，只是激活方式用了批量许可密钥。无病毒、无后门。

**❓ 可以更新吗？**  
建议关闭自动更新。开启更新可能导致激活失效。如果你想更新，更新后重新激活即可。

**❓ 支持哪些 Windows 版本？**  
Windows 10 和 Windows 11（64 位）。Windows 7/8 可以用旧版 Office。
